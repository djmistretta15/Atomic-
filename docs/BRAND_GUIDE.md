## Specimen Card System

Every product includes a **Specimen Card**‚Äîa scientific label printed on the garment or included as a physical insert.

### Required Elements

1. **Specimen Code** (e.g., `SPEC-LI-DIATOM-001`)
2. **Common Name** (e.g., "Diatom Colony")
3. **Scientific Name/Composition** (e.g., *Bacillariophyceae* spp., SiO‚ÇÇ)
4. **Technique** (e.g., "Brightfield Microscopy, 400x")
5. **Location** (e.g., "Long Island Sound, NY")
6. **Contributor** (e.g., "Dr. Sarah Moore, Marine Lab")
7. **QR Code** ‚Üí `/impact/{SKU}`

### Visual Style

- White text on dark overlay (primary navy at 90% opacity)
- Monospace font for technical data
- Grid background (subtle)
- Logo in corner

## Photography & Imagery Guidelines

### Product Photography

- **Background:** Solid off-white or gradient
- **Lighting:** Soft, even, no harsh shadows
- **Models:** Diverse, casual poses, minimal styling (let the garment be the hero)
- **Context shots:** Lab environments, beaches, campus, natural settings

### Specimen Imagery

- **Always original:** No stock microscopy
- **High resolution:** Minimum 300 DPI for print
- **Unmanipulated:** Color correction allowed, but no artificial enhancements
- **Credited:** Every image links to contributor

### Video

- **Zoom sequences:** Start wide, zoom into microscopic detail
- **Lab B-roll:** Researchers at work, microscopes in action
- **Impact footage:** Students using funded equipment, beach cleanups
